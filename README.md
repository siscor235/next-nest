# NestJS Playground

## Usage

```sh
docker-compose -up
dc exec web npm run console -- seed
```

http://localhost:3000/graphql

### Useful commands

Nest CLI:
```
dc exec web npm run nest -- --help
```

TypeORM CLI:
```
dc exec web npm run typeorm -- --help
```
